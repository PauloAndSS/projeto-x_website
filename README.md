# projeto-x_website
Site de busca e montagem de Decks para o jogo "Projeto-x" (Maycon Guedes Cordeiro). Desenvolvido por Paulo André Soares da Silva como trabalho de Back-end II do curso de Tecnologia em Sistemas para Internet (TSI) do IFES - Campus Santa Teresa, a plataforma otimiza a pesquisa e a criação de Decks para os jogadores.
